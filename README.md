# WebMidiOrbita

## Updating

### Update process

1. Clone this repository:

```bash
git clone https://github.com/Playtronica/WebMidiOrbita
```

2. Change the directory:

``` bash
cd WebMidiOrbita
```

3. Modify the contents of this directory. 
4. Sync the repository:

```bash
git commit -m "Description of changes"
git push
```
### Update Requirements:

- Do not rename the index.html.
- Use only relative href.
- Check your code for errors before saving to avoid issues.
- Changes will only be applied from the main branch.
